
    What is the purpose of a breakpoint?
    To stop the code from running at a certain point and run the debugging tool.

    What does the step into command do?

    What is the difference between evaluate expression and evaluate code fragment?
    You ca run multiple lines of code and methods using the evaluate code fragment but just one with the evaluate expression.
